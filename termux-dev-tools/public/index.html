Installation

# Clone the repository
git clone https://github.com/yourusername/termux-dev-essentials.git

# Navigate to directory
cd termux-dev-essentials

# Install dependencies
npm install

# Run setup
npm run setup

Quick Start

SMS Manager Example

const AndroidMessageManager = require('./src/sms-manager');

// Initialize with Twilio
const sms = new AndroidMessageManager('twilio');
sms.initTwilio('ACCOUNT_SID', 'AUTH_TOKEN');

// Send SMS
await sms.sendMessage('+1234567890', 'Hello from Termux!');

// Initialize with Vonage
const sms2 = new AndroidMessageManager('vonage');
sms2.initVonage('API_KEY', 'API_SECRET');
await sms2.sendMessage('+1234567890', 'Hello from Vonage!');

Environment Setup

Create a .env file with the following variables:

# SMS Provider Credentials
TWILIO_ACCOUNT_SID=your_twilio_account_sid
TWILIO_AUTH_TOKEN=your_twilio_auth_token
VONAGE_API_KEY=your_vonage_api_key
VONAGE_API_SECRET=your_vonage_api_secret

# GitHub Integration
GITHUB_TOKEN=your_github_token

# Development Settings
NODE_ENV=development
PORT=3000
